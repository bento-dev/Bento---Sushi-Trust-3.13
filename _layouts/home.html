The Bento-dev project in few words

The present project aims to gather developers interested in creating a gui software to ease the configuration of the different parts in an Openbox only desktop environment.

Bento is a distribution built on Ubuntu LTS, around the Openbox window manager and a few more components meant to form an easy and light environment for all users.

In order to pursue the aim of making it an easy almost-a-desktop environment we need to create a graphical user interface for an easy configuration of each part.

The resulting project should be available for all users who wish to setup their own Openbox environment, in any distribution
